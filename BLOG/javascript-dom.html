Control Flow and Loops in everyday life:

Control Flow is nothing but a sequence of steps execution in a particular order to fulfill the requirement We can take dough preparation (for Pizza) as an example and below are the steps resembles the control flow

1. Take the flour and weigh off 300 grams using the weighing scale.
2. Take the flour and add it to the bowl.
3. Add the yeast, salt, and water to it.
4. Start to knead the mixture.
5. During this process add olive oil to the dough, until it gets smooth and a little moist.
6. Knead it for about 5 minutes. This way the ingredients get well mixed and it will be smoother. 7. Once finished put it back in the bowl and cover it with a cloth.
Place the covered dough on a warm spot (heater for example) and leave it there for about 45 minutes.

Following the above sequence of instructions we can prepare dough this is nothing but following a control flow of instructions

LOOPS:-

Loop means particular function or statements executing continuously in java we can use 'for, while and do while' loops also there are 2 types of loops as conditional and unconditional

Below car wash example helps us to understand what exactly the LOOP means. The following sequence of steps are performed to complete a single car wash

1. Park the car
2. Vacuum the interior
3. Wax the car
4. Wash the car
5. Dry the exterior

Now let's assume a robot has to wash five cars and he will be repeating the sequence of above steps for each car wash so this called conditional loop

below is the explanation
for (C=1;C<=5;C+1);
{ . Park the car
. Vacuum the interior
. Wax the car
. Wash the car
. Dry the exterior
}
1. we ask the robot to start with one car (c=1)
2. Continue to do all tasks till he/she washes 5cars(c<=5)
3. Washes one car at atime(c+1)
4. Stops after 5 cars.

If we ask the robot to wash as many cars he/she needs to repeat the steps for unlimited number of times this is called unconditional loop the robot will keep repeating the steps for every car and will not stop until we program to stop with a break statement. In every programming language it is referred as while and do while loops

what functions are and why they are useful:

Function is a block of code designed to perform a particular task. They are useful as the function allows us to define and structure a block of code, give it a name, and helps us make the code reusable Example would be a wearing a ‘cloth mask’ could be reusable after wash rather than using a ‘one time use’ mask

In JavaScript, a function allows you to define a block of code, give it a name and then execute it as many times as you want. JavaScript function can be defined using function keyword. A function can have one or more parameters, which will be supplied by the calling code and can be used inside a function. JavaScript is a dynamic type scripting language, so a function parameter can have value of any data type.
